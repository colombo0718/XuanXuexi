# 宣學習 XuanXuexi

**LeafLune 宇宙** 旗下的漢語學習品牌，致力於以「寓教於樂」的方式讓語言學習更有趣、更有效。

---

## 產品線

### 🃏 Pinyin Poker 64（拼音撲克 64）
64 張涵蓋全部漢語拼音音素的撲克牌，4 個花色（聽力・口說・閱讀・書寫）× 16 張。

- 每張卡包含：主音、漢字、拼音、Emoji
- 適合：初學漢語的學習者、親子桌遊、課堂教具
- 格式：數位下載 PDF（自行列印）/ 未來實體印刷版

### ⚔️ Sentence Summoner（造句召喚戰）
用「造句」打架的漢語對戰卡牌遊戲。

- 詞性 = 卡片類型（名詞/動詞/形容詞/副詞/陷阱）
- 漢語程度決定你的戰力
- 支援 PvP（學生 vs 學生）與 PvE（全班 vs 老師 Boss）

### 📚 Chinese Flashcard
對齊 HSK 3.0 的主題式漢語學習卡，三維分類（難度/場景/文法）。
