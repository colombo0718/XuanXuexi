# LeafLune 卡牌／桌遊產品輸出標準 v1.0

> 目標：統一 LeafLune 品牌所有「卡牌／親子勞作桌遊」類產品的**最終輸出格式**，  
> 讓少人力的小團隊也能穩定產出「好印、好用、好溝通」的數位版與實體版。

---

## 0. 彩色與黑白列印策略

- **預設只製作彩色版本**，讓使用者在印表機選擇「灰階／黑白列印」即可。
- 僅在下列情況考慮另外製作「省墨黑白版」：
  - 遊戲或學習機制**高度依賴顏色**辨識。
  - 目標使用者多為「大量印」的學校或機構，有明確節省墨水需求。

黑白省墨版為**加值選項**，非所有產品必備。

## 1. 專案目錄結構（建議）

以「PinyinPoker」為例：

```text
PinyinPoker/
  src/                ← 網頁程式、原始設計檔
  export/
    digital/          ← 給 Gumroad 的數位版
    print_press/      ← 專給印刷廠的實體版檔案
  docs/
    product_spec.md   ← 產品規格 & 印刷說明（本文件的專案版本）
    readme_zh.pdf     ← 使用說明（給家長 / 老師）
    readme_en.pdf     ← （如有國際版需求）
  marketing/
    cover_gumroad.png ← 商品封面
    preview_cards.png ← 示意圖
```

每個新卡牌／桌遊專案，可複製此結構作為起點。

## 2. 數位版（Gumroad）輸出標準

> 目標：**給家長、老師的「好印好用版」**。  
> 不要求懂印刷，只要能打開 PDF、按列印、剪一剪就能使用。

### 2.1 必備檔案

放在 `export/digital/` 資料夾：

1. **主檔：彩色列印版 A4 PDF**

   - 檔名建議：`PinyinPoker_A4_color.pdf`
   - 規格：
     - 多頁 A4，直式（210 x 297 mm）。
     - 每頁排 8 張卡（2 x 4 格），或依實際設計調整。
     - 卡片外框或淡裁切線清楚，方便沿線剪裁。
   - 目標體驗：
     - 一般使用者只需下載本檔，直接列印即可。

2. **單張高解析 PNG**

   - 資料夾：`cards_png/`
   - 規格建議：
     - 長邊 ≥ 1200 px。
     - 解析度概念上對應 300 dpi（方便進一步編排或投影）。
   - 命名規則範例：
     - `A1_b.png`、`B6_a.png`、`C3_sh.png`……依產品邏輯統一命名。
   - 用途：
     - 進階使用者自行排版。
     - 之後 LeafLune 教材、簡報、網站展示。

3. **（可選）黑白 / 省墨版 A4 PDF**

   - 檔名建議：`PinyinPoker_A4_bw.pdf`
   - 僅在有明確省墨需求時製作：
     - 移除大面積底色。
     - 保留線條和文字，確保可讀性。

4. **使用說明與授權說明**

   - 檔名建議：`01_請先閱讀_使用說明.pdf`
   - 推薦內容：
     - 列印建議：
       - 列印設定：「實際大小」「不要縮放」。
       - 建議紙張厚度（例：120g 以上影印紙、或 200g 以上銅版紙）。
     - 剪裁建議：
       - 沿外框或裁切線剪下。
       - 是否建議護貝。
     - 簡單玩法示例：
       - 至少 2–3 種基本玩法（翻牌記憶、配對、造詞／造句等）。
     - 授權條款：
       - 允許教師／家長重複列印給學生使用。
       - 禁止轉售、上傳到其他素材平台公開散佈。

### 2.2 給 Gumroad 商品頁的標準敘述（範例）

> ✅ 內含：  
> ・彩色列印版 A4 PDF（多頁，直接印出剪下即可使用）  
> ・單張卡片高解析 PNG 圖檔  
> ・中文使用說明與簡單遊戲玩法  
>  
> 一般使用者只需下載並列印 PDF 檔案；  
> 若需自訂排版或製作簡報，可使用附贈的 PNG 圖檔。

所有數位商品應盡量用類似文字，明確告訴使用者「**預設用 PDF 就好**」。

## 3. 實體版（印刷廠）輸出標準

> 目標：**降低印刷溝通成本與出錯率**，  
> 讓不同印刷廠都能依據相同規格製作成品。

### 3.1 LeafLune 卡牌標準規格（LeafLune Card Standard v1）

建議作為 LeafLune 品牌預設卡牌規格，除非專案有特殊需求才修改。

- 成品尺寸（Finished size）：
  - **63 x 88 mm**（標準 Poker size）
- 出血（Bleed）：
  - 四邊各 **3 mm**
  - 設計尺寸 → **69 x 94 mm**
- 解析度（Resolution）：
  - **300 dpi**（或更高）
- 顏色模式（Color mode）：
  - **CMYK**（給印刷廠使用）
- 安全區（Safe area）：
  - 重要文字與關鍵圖案應離裁切線至少 **3 mm**。
- 圓角（Corner radius）：
  - 優先使用印刷廠標準撲克牌圓角模具。
  - 如需特殊圓角，需在 `product_spec.md` 特別標注。

### 3.2 印刷檔案格式

放在 `export/print_press/` 資料夾：

1. **正面（Fronts）**

   - 檔名建議：`fronts_300dpi_CMYK.pdf`
   - 形式：
     - 多頁 PDF，每頁一張卡的正面圖，含出血。
     - 或依印刷廠要求改為大版面拼版。

2. **背面（Backs）**

   - 檔名建議：`back_300dpi_CMYK.pdf`
   - 一張卡背圖案（含出血），由印刷廠重複拼版。

3. **技術要求**

   - 顏色：全部轉為 CMYK。
   - 解析度：300 dpi 以上。
   - 文字：全部轉曲線（outline），避免字型缺失。
   - 檔案：採用高品質 PDF（例如 PDF/X-1a）或印刷廠指定格式。

### 3.3 與印刷廠溝通範例說明

可寫入 `docs/product_spec.md`，並在洽談時直接引用：

> 我這裡有一套撲克牌尺寸的教具卡片：  
> - 成品尺寸：63 x 88 mm，四邊出血 3 mm（設計尺寸 69 x 94 mm）  
> - 正面共有 64 款不同圖案，背面統一 LeafLune 卡背  
> - 檔案為 300 dpi、CMYK、多頁 PDF，包含出血範圍  
> - 請使用貴公司標準撲克牌圓角裁切  
> - 推薦紙材可依一般撲克牌手感提供（例如 300–350g 霧銅，雙面霧膜）

此段可複製到每個專案的 `product_spec.md`，視個案微調數量與內容。

## 4. 黑白 / 省墨版政策

- 預設**不強制**製作省墨版。
- 當專案符合以下任一條件時，考慮新增：
  - 目標族群多數為學校／補習班，長期大量列印。
  - 遊戲機制可以在不依賴顏色的情況下運作。
- 省墨版設計原則：
  - 移除大面積背景填色。
  - 儘量利用線條、框線與簡單圖示維持可用性。
  - 保留 LeafLune 品牌識別（可用線條版 logo）。

## 5. 未來專案使用流程（Quick Checklist）

### 5.1 數位版（Gumroad）

- [ ] 所有卡面內容已定稿（文字、拼音、圖案、logo）。  
- [ ] 程式一鍵產生：
  - [ ] `XXX_A4_color.pdf`（彩色列印版）  
  - [ ] `/cards_png/` 內的單張高解析 PNG  
- [ ] 編寫並輸出：
  - [ ] `01_請先閱讀_使用說明.pdf`（包含列印、剪裁、授權、玩法）  
- [ ] 打包為 zip 上架至 Gumroad，商品頁文字包含：
  - [ ] 內容物說明  
  - [ ] 使用方式（預設用 A4 PDF）  
  - [ ] 授權範圍簡述  

### 5.2 實體版（印刷）

- [ ] 專案確認採用 LeafLune Card Standard v1 或明確標註例外。  
- [ ] 使用程式輸出：
  - [ ] `fronts_300dpi_CMYK.pdf`（含出血）  
  - [ ] `back_300dpi_CMYK.pdf`（含出血）  
- [ ] 確認：
  - [ ] 顏色已為 CMYK。  
  - [ ] 解析度為 300 dpi 以上。  
  - [ ] 文字已轉曲線。  
- [ ] 準備 `docs/product_spec.md` 給印刷廠，內容包含尺寸、紙材建議、數量與其他要求。
